## 几个你可能正在经历的瞬间

**你正在看一个加盟项目**
业务员给你看了官网、看了样板店、给你算了账。一切看起来都很正规。
但你总觉得哪里不对。
→ 快招公司最擅长的，就是让你觉得「看起来很正规」。

**你已经投了钱，店刚开起来**
前两周生意还行。然后突然就没人来了。
你开始怀疑选址，开始怀疑品类，开始怀疑自己。
→ 开业前两个月的营业额，要打个七折才是真实水平。

**有人劝你「再坚持一下」**
「餐饮都有养店期」「过了淡季就好了」「你这个位置其实不错的」。
→ 连续亏损超过 6 个月，90% 的情况下，坚持只会让你亏更多。

**你想接手一个转让的店**
对方说客流很好、盈利稳定、回头客多。
→ 转让费评估三件事：看后台流水、蹲点三天、问清楚对方为什么要转。

## 这套 Skill 能做什么

**在你掏钱之前**
选址对不对、毛利够不够、这个加盟商靠不靠谱——直接给你一个判断框架。不替你做决定，但让你知道该问什么问题。

**在你已经亏损的时候**
帮你快速诊断：是选址的问题、品类的问题、还是运营的问题。方向找对了，止损才有意义。

**在你被快招骗了之后**
维权三步走：固定证据、发函施压、起诉。拖得越久，钱越难要回来。

## 快招公司是怎么一步步套住你的

**第一步**
你在网上搜某个大品牌。弹出一个页面，说官方授权、区域限定、名额有限。留了电话，对方说「我们这里是唯一还开放加盟的渠道」。

**第二步**
被邀请去总部考察。写字楼里有一个精心装修的样板间。有人在现场签约，销售在旁边催促：「今天签能省 5 万，明天就截止了。」

**第三步**
签完合同之后，运营老师说：「建议你开在这里。」——把店开在品牌大店旁边，吃你的房租差价。三个月后撑不住，对方说：「是你的经营问题。」

整个过程，你没有任何一步是被强迫的。
但每一步，都有人精心设计过。
