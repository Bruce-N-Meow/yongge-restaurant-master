## 快招公司四步收割法

**第一步：截流 + 蹭大牌**
话术：「蜜雪不开放了，我们是他们的子品牌。」

**第二步：现场签约 + 托配合**
约到写字楼样板间，没有实体店，现场有托，制造紧迫感。

**第三步：二次收割——推代理**
「加 3 万拿代理，别人加盟你拿 70%。」实际上根本没人加盟。

**第四步：选址收割 + 推卸责任**
选址老师故意把店开在品牌大店旁边，吃房租差价。店亏了就是「你经营的问题」。
