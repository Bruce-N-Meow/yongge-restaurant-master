## 防骗核心：快招公司四步收割法

**第一步：截流 + 蹭大牌**
截留想加盟大品牌的人，话术：「蜜雪这边不开放了，但我们是他们的子品牌」

**第二步：现场签约 + 托配合**
约到总部写字楼样板间（没有实体店），现场有托，制造紧迫感

**第三步：二次收割——推代理**
「加3万拿代理，以后别人加盟你拿70%加盟费」，实际上根本没人会加盟

**第四步：选址收割 + 推卸责任**
选址老师故意选在品牌大店旁边，吃房租差价，店亏了就是「你经营的问题」
